# Taraba Tourism Portal 🌍

> A responsive tourism web portal implementing advanced CSS architectures.

**🔴 [View Live Demo](https://haipedia.github.io/travel-web)**

## About The Project
A front-end web application designed to promote tourism in Taraba State. This project focuses heavily on complex image gallery layouts, alternating visual pacing, and modern booking interface design.

## Tech Stack
* **HTML5:** Semantic web structure and accessibility.
* **CSS3:** Advanced CSS Grid, Flexbox, and custom styling.

## Key Features
* Complex masonry image gallery built entirely with CSS Grid.
* Alternating UI layouts for engaging visual storytelling.
* Clean, responsive adventure booking form UI.
